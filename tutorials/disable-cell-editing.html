Not editable columns

In many cases, you will need to configure a certain column to be not editable. Making columns not editable doesn’t change it’s behaviour. It is still available for keyboard navigation, CTRL+C and CTRL+V functionalities and so on.

To make a column not editable, declare it in the columns setting. You can also define a special renderer function that will dim the editor value.
