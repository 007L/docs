Our philosophy at Handsoncode comprises of the seven principles shared below:

  1. Performance is the our number one priority,
  2. We aim to provide a bug free code, easy to maintain and extend,
  3. We improve the code quality with patch releases every two weeks,
  4. We test each release against recent versions of the most popular browsers,
  5. We write well documented code,
  6. We write testable code,
  7. We follow Airbnb JavaScript style guide.
